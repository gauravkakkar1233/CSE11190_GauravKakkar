# Title                                                                 BY

Campaign House – Campaign Management System                              ~ Gaurav Kakkar

# Description

Campaign House is a full-stack campaign management application where users can register, login, create, search, and manage campaigns.

The project uses **React + TypeScript** for frontend and **FastAPI with MongoDB** for backend. It demonstrates REST API development, CRUD operations, authentication, database connectivity, API testing, and MongoDB-based data management.

# REST API Modules

## User Management

POST   /register

POST   /login

GET    /users

PUT    /user/{id}

DELETE /user/{id}

## Campaign Management

POST   /campaign

GET    /campaigns

GET    /campaign/{id}

PUT    /campaign/{id}

DELETE /campaign/{id}

## Notification Management

POST /notification

GET  /notifications

## File Upload

POST /upload
